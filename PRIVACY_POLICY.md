# Política de Privacidad - Botón de Pánico

**Última actualización:** Abril 2026

## 1. Introducción

La aplicación "Botón de Pánico" (en adelante, la "Aplicación") está diseñada para proporcionar funciones de emergencia que permitan compartir la ubicación del usuario con contactos de emergencia seleccionados.

## 2. Datos Recopilados

La Aplicación recopila y procesa los siguientes datos:

### 2.1 Ubicación
- **Coordenadas GPS**: Se recopila la ubicación precisa del dispositivo cuando se activa el botón de pánico
- **Propósito**: Compartir la ubicación con contactos de emergencia
- **Frecuencia**: Cada 5 segundos durante 8 horas (periodo de pánico)

### 2.2 Contactos
- **Números de teléfono**: Se solicitan números de contactos de emergencia
- **Propósito**: Enviar mensajes SMS con información de ubicación

### 2.3 Datos del Dispositivo
- Versión de Android
- Permisos otorgados

## 3. Uso de Datos

Los datos recopilados se utilizan exclusivamente para:
- Compartir ubicación con contactos de emergencia
- Enviar mensajes SMS con coordenadas GPS
- Mejorar la funcionalidad de la aplicación

## 4. No Compartimos Datos

- **No compartimos datos** con terceros
- **No vendemos datos** personales
- **No usamos análitica** de seguimiento

## 5. Seguridad

Los datos se almacenan localmente en el dispositivo del usuario. No se envían a servidores externos excepto mediante SMS a los contactos seleccionados por el usuario.

## 6. Retención de Datos

Los datos de ubicación se almacenan temporalmente durante el periodo de pánico (8 horas) y se borran automáticamente después.

## 7. Derechos del Usuario

El usuario tiene derecho a:
- Acceder a sus datos
- Eliminar sus datos
- Revocar permisos de la aplicación
- Desinstalar la aplicación en cualquier momento

## 8. Cambios en la Política

Nos reservamos el derecho de actualizar esta política. Los cambios se notificarán mediante actualización de la aplicación.
